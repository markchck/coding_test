1. 아직 못 풀었으면 a :  111_a.py
2. 다른 풀이 있으면 b :  111_b1.py (숫자가 높을수록 좋다고 생각되는 풀이)
3. 중요도에 따서 s, ss, sss
4. 다시 풀었을 때 과연 맞출 수 있을까 의심 들면 : r
5. 혼자 시간재면서 복기할 때 푼 풀이 : self
